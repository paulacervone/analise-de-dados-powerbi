# Análise de Dados e Dashboards de Desempenho 📊

## 💻 Sobre o Projeto
Este projeto tem como foco a recolha, limpeza, tratamento e visualização de dados. O objetivo principal foi transformar uma base de dados bruta em informações estratégicas, criando painéis interativos (Dashboards) para facilitar a monitorização de Indicadores-Chave de Desempenho (KPIs laboratoriais) e dar suporte à tomada de decisão.

## ⚙️ Etapas e Funcionalidades
- **Extração e Limpeza:** Tratamento de dados inconsistentes, remoção de duplicados e formatação de valores utilizando o Excel.
- **Modelagem de Dados:** Estruturação das tabelas e criação de relacionamentos no Power BI.
- **Criação de Medidas (DAX):** Desenvolvimento de cálculos personalizados para gerar os indicadores necessários (como o Tempo de Resposta/TAT e a Taxa de Rejeição).
- **Visualização:** Construção de gráficos interativos e painéis dinâmicos para análise intuitiva.

## 🛠️ Tecnologias e Ferramentas Utilizadas
- **Microsoft Excel:** Limpeza e tratamento inicial dos dados.
- **Microsoft Power BI:** Modelagem de dados, fórmulas DAX e criação dos Dashboards interativos.
- **Conceitos aplicados:** Business Intelligence (BI), Análise Exploratória de Dados (EDA) e Data Storytelling.

## 🚀 Como visualizar o projeto

1. Faça o download do ficheiro `.pbix` disponível neste repositório.
2. Abra o ficheiro utilizando o **Microsoft Power BI Desktop**.
